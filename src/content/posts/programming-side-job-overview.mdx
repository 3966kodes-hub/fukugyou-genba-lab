## 🚀 未経験から月 10 万円までの 4 ステップ

### ステップ 1：基礎学習（3〜6 ヶ月）

学習リソース：

| リソース | 費用 | 期間 |
|---|---|---|
| Progate / ドットインストール | 月 ¥1,078 | 1〜2 ヶ月 |
| Udemy 動画講座 | 各 ¥1,500〜 | 2〜3 ヶ月 |
| プログラミングスクール | ¥200,000〜700,000 | 3〜6 ヶ月 |
| 独学（書籍 + 公式ドキュメント）| ¥10,000〜 | 6 ヶ月〜 |

→ **コスト重視なら Progate + Udemy**、**時短狙いならスクール**。

### ステップ 2：ポートフォリオ作成（1〜2 ヶ月）

「自分が何を作れるか」を **3〜5 個の作品**で示す。

ポートフォリオ例：
- 自分の HP（GitHub Pages or Netlify ホスティング）
- 架空店舗の LP（HTML/CSS のみ）
- WordPress カスタマイズ事例（オリジナルテーマ）
- 簡易 Web アプリ（タスク管理など）

→ GitHub アカウントで公開 + ポートフォリオサイトに掲載。

### ステップ 3：初案件獲得（1〜3 ヶ月）

クラウドソーシング（クラウドワークス・ランサーズ）で：

1. プロフィールを充実（実績・スキル・ポートフォリオリンク）
2. 「コーディング初心者歓迎」案件に 30 件以上応募
3. **低単価でも初案件を取って実績作り**（¥3,000〜5,000 でも OK）
4. 評価 ⭐⭐⭐⭐⭐ 獲得

### ステップ 4：継続案件化 + 単価アップ（6 ヶ月〜）

- 単発案件 → 継続案件への移行交渉
- 単価交渉（実績増えたら ¥10,000 → ¥30,000 へ）
- 直営業（SNS・知人経由）でクラウドソーシング手数料カット

## 🤖 AI 時代のプログラミング副業

GitHub Copilot・Cursor・Claude Code 等の AI ツールで **「初心者でもコードが書ける」**時代に。

**生き残り戦略：**

1. **AI を使い倒して生産性を 2-3 倍化** → 単価アップ交渉
2. **要件定義・設計の上流工程**（AI ができない領域）
3. **特定ドメイン特化**（医療系・金融系・物流系）
4. **保守・運用の継続案件**（AI では完結しない領域）

## まとめ：長期投資前提なら最強の副業

プログラミング副業は **「学習期間 1 年の覚悟があれば、月 30 万円超も視野」**の優良ジャンル。短期収益狙いには向きません。
