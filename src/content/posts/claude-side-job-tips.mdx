Anthropic の Claude は ChatGPT と並ぶ生成 AI の双璧。特に **「長文処理・コーディング・調査」**で頭一つ抜けています。本記事で副業活用 10 例を解説。

## 🎯 Claude が ChatGPT より優れている領域

| 領域 | Claude の優位性 |
|---|---|
| **長文処理** | 200K トークン（約 15 万字）入力対応 |
| **コーディング** | Claude Code・Cursor で開発 AI 最強 |
| **調査・分析** | 引用元保持・根拠説明が丁寧 |
| **倫理的配慮** | 不適切内容の排除がより厳格 |

## 💡 具体的活用 10 例

### ① 長文記事の一括校正

**プロンプト：**
```
以下の 8,000 字記事を校正してください。
- 誤字脱字
- 表現重複
- 読みやすさ
- SEO 観点での改善

[8,000 字の記事全文]
```

→ ChatGPT は分割必要 / Claude は一括処理 OK。

### ② Kindle 本の構成案 + 章ごと執筆

**プロンプト：**
```
「副業ポートフォリオ構築術」のKindle 本構成を作成してください。
- 全 12 章
- 各章 2,500 字目安
- 各章に H2/H3 設計
- 全体の論理整合性チェック
```

→ 30,000 字超の Kindle 本企画を 1 時間で構成完了。

### ③ プログラミング副業（Web 制作）

Claude Code で：
- WordPress テーマ修正
- React コンポーネント実装
- バグ修正・リファクタ

→ 案件納期短縮 → 単価据置で収益倍増。

### ④ 大量 PDF / 資料の要約

**プロンプト：**
```
以下の業界レポート（200 ページ PDF）を要約してください。
- 経営者視点で重要なポイント 10 個
- 図表は箇条書きで再構成
- 引用ページ番号併記
```

→ 副業の調査時間 1/10 に。

### ⑤ 競合サイト全記事の比較分析

**プロンプト：**
```
以下の競合サイト 3 つの記事リスト（各 30 記事）を分析し、
共通テーマ / 抜けている観点 / 独自性のある記事を整理してください。
```

→ 競合分析の精度 + 速度が圧倒的。

### ⑥ ASP 案件の比較表生成

**プロンプト：**
```
以下の ASP（A8.net・もしも・バリュコマ）の主要 20 案件を
ジャンル別に整理し、最高単価ランキング表を作成してください。
```

→ ASP 戦略立案が瞬時に。

### ⑦ メールテンプレート生成

**プロンプト：**
```
クライアントへの単価交渉メール、
依頼お礼メール、
納品報告メール、
の 3 種類を、丁寧な敬語で作成してください。
```

→ 営業メール時短化。

### ⑧ Excel / スプレッドシート関数生成

**プロンプト：**
```
以下の表で、A 列の値が「公開済」のとき
B 列の合計を求める Excel 数式を教えてください。

[表データ]
```

→ Excel 苦手な人でも即解決。

### ⑨ 動画台本 / YouTube 構成

**プロンプト：**
```
「副業ブログで月 5 万達成」をテーマに、
10 分の YouTube 動画台本を作成してください。
- 冒頭 30 秒で離脱防止のフック
- 章立て + ナレーション原稿
- 終わりにチャンネル登録誘導
```

→ YouTube 副業の構成設計が楽に。

### ⑩ 確定申告の事前チェック

**プロンプト：**
```
以下の収支データから、副業の所得（雑所得）を計算し、
青色 / 白色どちらが節税効果が大きいか判定してください。

[収支データ]
```

→ 税理士相談前の事前検討が深まる（ただし最終判断は税理士に）。

## 🔧 Claude Code（開発者向け）

CLI ツールとして提供される **Claude Code** は、副業エンジニアに革命をもたらしています：

- ターミナルから直接コーディング
- ファイル編集・実行可能
- 大規模リファクタリング
- バグ修正・テスト生成

→ 副業の Web 制作案件を **半分の時間で完了**できる事例多数。

## ⚠️ 注意点

<Callout type="warning" title="Claude を使う上での注意">
1. **倫理ガード厳しめ** → 際どい内容は ChatGPT より制限される
2. **画像生成なし** → 画像は ChatGPT 4o or Midjourney 併用
3. **一部機能 ChatGPT 優位** → ジャンルで使い分け推奨
4. **AI 量産記事は Google ペナルティ** → 人間レビュー必須
5. **機密情報入力禁止** → クライアント情報は入れない
</Callout>
