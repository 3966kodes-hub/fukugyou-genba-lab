メルカリ副業は **「ゼロ円で始められる × 即金性 × 学習コスト低」**で副業初心者の入口として最適。本記事で月 1 万 → 5 万 → 10 万への 3 段階進化戦略を解説します。

## 💰 メルカリの仕組み（基本）

| 項目 | 値 |
|---|---|
| 出品手数料 | 無料 |
| 販売手数料 | **10%**（販売価格の） |
| 振込手数料 | ¥210（¥10,000 以上は無料）|
| 出金可能額 | ¥10,000 以上推奨 |
| メルカリ Shops | 別ストア（手数料 10% + 月額 ¥0）|

→ シンプルな手数料体系。¥1,000 売って手取り ¥900。

## 🚀 3 段階の進化戦略

### 段階 1：不用品販売（月 1〜3 万円）

**期間：** 3 ヶ月

家にある不用品を 30〜100 点出品 → メルカリ販売の流れを学ぶ。

**売れやすい不用品：**
- ブランド服 / バッグ
- 家電 / ガジェット
- 本 / 漫画 / DVD
- ベビー用品（特に未使用品）
- 限定グッズ / ノベルティ

### 段階 2：物販副業（月 3〜10 万円）

**期間：** 3〜12 ヶ月

不用品が尽きたら **「仕入 → 販売」のせどりモデル**に転換。

**仕入先：**
- ブックオフ / セカンドストリート（中古）
- ドンキ / しまむら / 100 均（季節商品）
- 楽天スーパーセール（電脳せどり）
- メーカー直販の限定セール

<Callout type="warning" title="メルカリ × 古物商">
中古品を仕入れて販売する場合、**古物商許可**が必要なケースがあります。継続的に営利目的で中古品売買するなら、最寄り警察署で古物商許可を申請してください。
</Callout>

### 段階 3：ハンドメイド / 自社製品販売（月 10 万〜）

**期間：** 1 年〜

自分で作った商品 / 自社ブランド商品を販売。**メルカリ Shops** に切替推奨。

**展開例：**
- ハンドメイド作家化（minne / Creema 併用）
- セレクトショップ風の自社商品ラインナップ
- 海外輸入の独自商品（中国輸入 / 韓国輸入）

## 📊 売れる出品の 5 大要素

### ① 写真の質

**最初の写真（サムネイル）で決まる**：
- 自然光 + 白背景
- 商品の正面 + 角度違いで 4〜10 枚
- 汚れ / 傷を正直に撮影（クレーム回避）

### ② タイトル（40 文字以内）

検索ヒット重視：
- ブランド名 + 商品名 + サイズ + 状態
- 例：「Nike エアフォース 1 27.5cm 美品 メンズ スニーカー」

### ③ 説明文

含めるべき情報：
- 商品状態（新品 / 美品 / 使用感あり）
- サイズ / 素材
- 使用期間
- 発送方法 + 期間
- ペット / 喫煙の有無（クレーム防止）

### ④ 価格設定

メルカリ独自の **「相場リサーチ機能」**を活用：
- 類似商品の **販売済み平均価格**で設定
- 高すぎても売れず、安すぎると赤字

### ⑤ 出品タイミング

- **平日 20-22 時 / 週末 10-15 時**が閲覧ピーク
- 出品から 24 時間以内に売れることが多い

## 🛡 メルカリ副業のリスク管理

| リスク | 対策 |
|---|---|
| アカウント BAN | 規約遵守 / 1 アカウントのみ運用 |
| 返品・低評価 | 商品状態の正直記載 + 検品徹底 |
| 個人情報漏洩 | らくらくメルカリ便（匿名配送）使用 |
| 偽物・違法品 | ブランド品の真贋証明 / 取扱禁止品の確認 |
| 古物商トラブル | 継続的中古売買は古物商許可取得 |

## まとめ：副業初心者の入口として最強

メルカリ副業は **「失敗してもリスクほぼゼロ」**で、まず副業を体験するのに最適。不用品 30 点を 1 ヶ月で売り切ったら、物販副業への進化を検討しましょう。
