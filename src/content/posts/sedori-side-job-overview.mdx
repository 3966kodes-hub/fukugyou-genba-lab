せどり副業は **「即金性 + 学習コスト低 + ROI 計算しやすい」**で根強い人気。本記事で 3 モデルの違い + 副業として成立する見極めを解説します。

## 🛒 せどり 3 モデル比較

| 項目 | 店舗せどり | 電脳せどり | Amazon FBA |
|---|---|---|---|
| **仕入先** | 実店舗（家電量販店・ブックオフ等）| ネット通販（メーカー直販・他 EC）| メーカー / 卸（大量仕入）|
| **販路** | メルカリ / Amazon / ヤフオク | 同左 | Amazon FBA |
| **必要資金** | ¥30,000〜 | ¥50,000〜 | ¥300,000〜 |
| **必要時間** | 移動 + 仕入で長め | 完全在宅 | FBA 倉庫が代行 |
| **利益率** | 20〜40% | 15〜30% | 10〜25% |
| **スケール性** | 🔴 低（時間限界）| 🟡 中 | 🟢 高（自動化） |
| **副業適性** | 🟡 週末型 | 🟢 在宅型 | 🟡 上級者向 |

## 🎯 各モデルの詳細

### ① 店舗せどり

**メリット：** 出品者が少ない掘り出し物を見つけられる / 仕入即購入できる

**デメリット：** 移動時間 + 重い商品の運搬

**狙い目：**
- 家電量販店の決算セール
- ブックオフのプレミア本
- ドンキ・しまむら等のレジャー商品

### ② 電脳せどり

**メリット：** 完全在宅 / スプレッドシート管理しやすい / 24h リサーチ可能

**デメリット：** 競合多数 / 価格変動激しい / 仕入価格の上昇

**狙い目：**
- メーカー公式 Amazon ストアの限定セール
- 楽天スーパーセール × ポイント還元
- ヤフーショッピング × PayPay 還元

**必須ツール：**
- Keepa（Amazon 価格履歴）
- せどりすと / プライスター（管理）

### ③ Amazon FBA（Fulfillment by Amazon）

**メリット：** Amazon 倉庫保管 + 出荷代行 / プライム表示で売れやすい

**デメリット：** 在庫保管料 + 出荷手数料 / 大量仕入が必要

**料金（参考）：**
- 在庫保管料：月 ¥10〜30 / 商品サイズ
- 出荷手数料：¥250〜500 / 件
- 月額（大口出品）：¥4,900

## 🚪 副業として成立する見極め基準

<Callout type="warning" title="副業として成立しない可能性が高いケース">
- 仕入資金 ¥30,000 未満で始める
- 月 20 時間以上を確保できない
- 在庫保管スペースがない（家族と共用）
- 物流オペレーションの管理ができない
</Callout>

副業せどりが成立する 5 条件：

1. 在庫資金 **¥100,000 以上** 用意できる
2. 月 **20〜40 時間** をリサーチ / 出品 / 発送に割ける
3. 在庫保管スペース **2〜3 m²** を家庭内で確保
4. 商品リサーチが好き（楽しめないと続かない）
5. 在庫リスクを許容できる（売れ残り 10〜20% は織り込む）
