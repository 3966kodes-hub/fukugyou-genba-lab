タイムチケットは **「30 分単位の時間を販売する」**独自モデルのスキルマーケット。本記事で活用法を整理します。

## 💰 タイムチケットの基本

| 項目 | 値 |
|---|---|
| 月額 | ¥0 |
| 出品手数料 | ¥0 |
| 販売手数料 | **20〜25%**（販売実績で変動）|
| 単位 | **30 分単位の時間を販売** |
| 最低単価 | ¥1,500 / 30 分 |
| 形式 | オンライン会議（Zoom 等）/ 対面 |

## 🎯 売れる「時間商品」例

| カテゴリ | 商品例 | 単価 |
|---|---|---|
| 相談・コンサル | 副業相談 / キャリア相談 | ¥3,000〜10,000 / 30 分 |
| スキル指導 | プログラミング指導 / 写真教室 | ¥3,000〜10,000 / 60 分 |
| 占い・スピリチュアル | 占い相談 | ¥2,000〜5,000 / 30 分 |
| 雑談・話し相手 | 異業種交流 / 雑談 | ¥1,500〜3,000 / 30 分 |
| 練習相手 | 英会話 / 面接対策 | ¥2,500〜5,000 / 30 分 |
| 同行・代行 | 街歩き案内 / 買い物同行 | ¥2,000〜5,000 / 60 分 |
| 専門家相談 | 税務 / 法律 / 医療相談 | ¥5,000〜30,000 / 30 分 |

## 🛒 ココナラとの違い

| 観点 | タイムチケット | ココナラ |
|---|---|---|
| 形式 | **時間販売（同期型）** | 成果物販売（非同期型） |
| 単価 | ¥1,500〜10,000 / 30 分 | ¥500〜100,000 / 件 |
| ユーザー数 | 約 50 万人 | 約 400 万人 |
| 競合度 | 🟢 低（ニッチ）| 🔴 高 |
| 副業適性 | 同期型 OK な人 | 非同期 OK な人 |

→ **「リアルタイム会話が苦じゃない」人はタイムチケット、「成果物納品が好き」人はココナラ**。

## 🎯 売れるチケット作成の 5 ポイント

### ① 具体的なターゲット設定

❌ 「副業相談乗ります」
✅ 「**会社員×副業ブログ初心者**向け：1 ヶ月で初記事公開する具体プラン作成」

### ② 成果の明確化

「30 分で **○○ ができるようになる**」と具体的なアウトプットを約束。

### ③ 時間設定

- 30 分：軽い相談 / 簡単な質疑応答
- 60 分：本格コンサル / 教室
- 90-120 分：ワークショップ / 集中相談

### ④ 価格設定

最初は **同ジャンル相場の 70-80%** で出品 → 評価獲得 → 段階的に値上げ。

### ⑤ サムネイル + タイトル

- タイトル：30 文字以内 + 具体的な数字
- サムネイル：顔写真 or 専門性アピール画像
