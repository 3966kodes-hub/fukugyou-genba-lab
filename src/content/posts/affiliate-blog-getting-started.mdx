「アフィリエイトブログ始めたいけど、何から手を付ければ？」— その答えは **「1 日でセットアップ完了」**です。本記事で完全手順を解説します。

## 🎯 全体フロー（所要時間：約 4 時間）

| ステップ | 所要時間 |
|---|---|
| 1. レンタルサーバー契約 | 30 分 |
| 2. ドメイン取得 | 15 分 |
| 3. WordPress インストール | 15 分（自動）|
| 4. テーマ選定・有効化 | 30 分 |
| 5. 初期設定（プラグイン・SEO 設定）| 1 時間 |
| 6. 自己紹介ページ作成 | 30 分 |
| 7. 初記事執筆・公開 | 1 時間 |

## ステップ 1：レンタルサーバー契約

### 主要 3 サーバー比較

| サーバー | 月額 | 速度 | 初心者向け度 |
|---|---|---|---|
| エックスサーバー | ¥990〜 | 🟢 高 | 🟢 |
| ConoHa WING | ¥990〜 | 🟢 最速級 | 🟢 |
| mixhost | ¥990〜 | 🟢 高（LiteSpeed）| 🟡 |

→ **初心者は ConoHa WING が一番カンタン**（WP 自動セットアップ + 速度 + UI が良い）。

### 契約手順

1. 公式サイトでプラン選択（**ベーシック**で十分）
2. 契約期間（**12 ヶ月で月 ¥990**）
3. ドメインも同時取得可能（**永久無料特典**）
4. クレジット決済 → 即時アカウント開通

## ステップ 2：ドメイン取得

| TLD | 価格 | 推奨度 |
|---|---|---|
| **.com** | ¥1,500/年 | 🟢 王道 |
| .net | ¥1,500/年 | 🟢 |
| .blog | ¥3,000/年 | 🟡 |
| .jp | ¥3,000〜 | 🟢 |

→ **ConoHa WING の永久無料特典で .com を選ぶ**のが最強。

ドメイン名のコツ：
- 短い・覚えやすい
- ジャンル想起しやすい単語
- 既存サイトと被らない

## ステップ 3：WordPress インストール

ConoHa WING の場合、**契約と同時に WordPress 自動セットアップ**機能あり：

1. 管理画面で「WordPress かんたんセットアップ」選択
2. サイト名・ユーザー名・パスワード設定
3. テーマ選択（後で変更可）
4. **5 分待つ → SSL 自動有効化 → サイト稼働開始**

## ステップ 4：テーマ選定

| テーマ | 価格 | 特徴 |
|---|---|---|
| Cocoon | 無料 | 軽量・SEO 強・初心者向け |
| Lightning | 無料 | シンプル・カスタマイズ性高 |
| **SWELL** | ¥17,600 | 国内最人気・ブロック編集に最適 |
| AFFINGER | ¥14,800 | アフィ特化・収益化機能豊富 |
| JIN:R | ¥19,800 | 美麗デザイン・初心者向け |

→ 副業ブログ本気でやるなら **SWELL or AFFINGER（有料）** 推奨。長期投資で回収。

## ステップ 5：初期設定（必須プラグイン）

<Callout type="tip" title="最低限入れるべきプラグイン">
- **SEO SIMPLE PACK**：SEO 設定一括
- **Contact Form 7**：問い合わせフォーム
- **XO Security**：セキュリティ強化
- **Akismet**：スパム対策
- **Site Kit by Google**：GA4 + Search Console 連携
</Callout>

その他設定：
- パーマリンク：`/posts/%postname%/`（記事 URL を綺麗に）
- 一般設定：サイトタイトル + キャッチフレーズ
- HTTPS リダイレクト確認

## ステップ 6：自己紹介ページ作成

「固定ページ」で：
- About / プロフィール
- お問い合わせ
- プライバシーポリシー
- 免責事項

→ Google AdSense 申請には **これら 4 ページが必須**。最初に作っておく。

## ステップ 7：初記事執筆・公開

最初の 5 記事のテーマ：

1. **柱記事 1**：ジャンル全体像（5,000 字以上）
2. **柱記事 2**：始め方ガイド（3,000 字以上）
3. **比較記事**：A vs B（3,000 字以上）
4. **体験記事**：自分の体験（2,000 字以上）
5. **問題解決記事**：困りごと → 解決策（2,000 字以上）
