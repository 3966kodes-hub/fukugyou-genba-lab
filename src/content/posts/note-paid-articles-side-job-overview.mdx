note は **「フォロワー × コンテンツ販売」**で個人クリエイターが収益化できる人気プラットフォーム。本記事で活用法を整理します。

## 💰 note の収益モデル 3 種

### ① 単発有料記事

1 記事を有料設定（¥100〜50,000）して販売。

| 項目 | 値 |
|---|---|
| 価格設定 | ¥100〜50,000 |
| 手数料 | クレジット決済 5% / 携帯決済 15-20% + プラットフォーム利用料 10% |
| 入金 | 月末締・翌月末払 |

→ **¥1,000 記事を 100 部販売 = ¥10 万から手数料 15-30% 引** = 実質 ¥7-8.5 万。

### ② マガジン（複数記事のまとめ売り）

複数の有料記事をパッケージ販売。

例：
- 「副業ブログ完全ガイド」マガジン（10 記事 ¥3,000）

### ③ メンバーシップ（サブスク）

月額制の継続課金。コミュニティ + 限定記事配信。

| 項目 | 値 |
|---|---|
| 月額設定 | ¥100〜10,000 / 月 |
| 手数料 | 同上 |
| 機能 | 限定記事 + 限定掲示板 + Slack 連携 |

## 🎯 売れる note 記事の 5 大要素

### ① タイトル

数字 + ベネフィット明確化：
- 「月 100 万円稼いだ会社員副業ブログ運用全工程」
- 「失敗から学ぶ Kindle 出版で売れる本の書き方 7 つ」

### ② 無料部分

冒頭 30〜50% は無料公開。**ここで「続きを読みたい」と思わせる**のが鍵。

### ③ 価格設定

| 価格帯 | 内容 |
|---|---|
| ¥100-300 | 集客用・気軽に試せる |
| ¥500-1,500 | 主力販売価格帯 |
| ¥3,000-10,000 | 専門性高 + 体系的内容 |
| ¥10,000-50,000 | 完全パッケージ + コンサル特典 |

### ④ 内容のオリジナリティ

- 自分の実体験
- 独自の数字データ
- 業界知識・裏話

### ⑤ ビジュアル

- アイキャッチ画像
- 記事内画像（5-10 枚）
- 図解・表

## 🚀 フォロワー獲得戦略

### ① 無料記事を継続投稿

- 週 1-2 本ペース
- フォロワー獲得目的の良質コンテンツ

### ② SNS 連携

- X（Twitter）でシェア
- プロフィールに note リンク

### ③ note 内交流

- スキ・コメント・フォロー
- 関連クリエイターとの相互フォロー

### ④ 初有料記事は無料配布

最初の 1-2 本は **無料配布キャンペーン**で読者の手に届ける → リピート期待。

## 🆚 note vs ブログ vs Kindle

| 観点 | note | ブログ | Kindle |
|---|---|---|---|
| 立ち上げ速度 | 🟢 即日 | 🟡 1 週間 | 🟡 数ヶ月 |
| デザイン自由度 | 🔴 低 | 🟢 高 | 🟡 中 |
| SEO 流入 | 🔴 弱 | 🟢 強 | 🟡 中 |
| ファン化 | 🟢 強 | 🟡 中 | 🟡 中 |
| 収益化 | 🟢 即可能 | 🔴 半年〜 | 🟡 出版後 |
| 単価 | 🟡 ¥100-50,000 | 🟢 ¥100/click〜 | 🟡 ¥99-3,000 |

→ **3 つを併用して最大化**するのが王道。

## 💡 note で稼ぐ人の共通パターン

1. **本業の専門性 × 体験談**を組み合わせる
2. **無料記事 → 有料記事への動線**設計
3. **SNS（特に X）でフォロワー獲得**を並行
4. **連続シリーズ化**でリピート購買誘導
5. **マガジン化 + サブスク**で収益安定化
